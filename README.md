# Crowdfunding Land SPA

Современное одностраничное приложение (SPA) для поиска и управления инвестициями в земельные участки. Проект построен на стеке React + Redux Toolkit с использованием кастомной сборки Webpack.

## Функциональные возможности

### Система авторизации и доступа

- **Умная сессия**: Сохранение JWT-токена и профиля в `localStorage`.
- **Разграничение прав**: Автоматический редирект и условный рендеринг: гостям доступен лендинг, авторизованным пользователям — полный функционал платформы.
- **Интеграция с API**: Полноценный цикл Auth (Login/Logout) через мутации RTK Query (DummyJSON API).

### Маркетплейс локаций (ShopPage)

- **Динамический каталог**: Загрузка списка объектов в реальном времени.
- **Система фильтрации**: Поиск по названию, стране или городу с мгновенным обновлением результатов.
- **Интеллектуальная сортировка**: Сортировка по алфавиту и географии, реализованная через `useMemo` для предотвращения лишних перерисовок.
- **Интерактивные карты**: Визуализация каждого объекта на карте с использованием `Leaflet`.

### Умная корзина (Personalized Cart)

- **Изолированное хранение**: Реализована логика раздельных корзин. Товары пользователя "А" никогда не отобразятся у пользователя "Б" благодаря динамическим ключам `cart_${userId}` в хранилище.
- **Persistence**: Состояние корзины сохраняется после закрытия браузера, но безопасно очищается при выходе (Logout).
- **Сводка заказов**: Автоматический расчет общей стоимости с использованием метода `reduce` и форматированием валют.

## Технологический стек

- **Core**: React 18 (Hooks, Lazy, Suspense)
- **State**: Redux Toolkit & RTK Query
- **Maps**: Leaflet & React-Leaflet
- **Styling**: Scoped CSS Modules
- **Build**: Custom Webpack Config (Babel, Asset Modules, DevServer)

---

## Основные зависимости

- `react`, `react-dom`, `react-router-dom` — ядро и навигация.
- `@reduxjs/toolkit`, `react-redux` — управление состоянием и серверный кеш.
- `leaflet`, `react-leaflet` — работа с геопозициями и картами.

## Инструкция по запуску

### 1. Требования

Убедитесь, что у вас установлен **Node.js** (рекомендуемая версия 16.x или 18.x) и менеджер пакетов **npm**.

### 2. Установка

Клонируйте репозиторий и установите все необходимые модули:

```bash
git clone [https://github.com/your-username/crowdfunding-spa.git](https://github.com/your-username/crowdfunding-spa.git)
cd crowdfunding-spa
npm install
npm run build
npm start
```
